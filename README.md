Movie List App (React Learning Project)

A simple and clean React-based Movie Listing & Filtering application built as part of my learning journey.
This project helped me practice React components, props, state management, filtering, conditional rendering, and basic UI design.

🚀 Features
✔️ List Movies

Displays a collection of movies with details such as:

Title

Year

Genre

Rating

Thumbnail (optional)

✔️ Filter Movies

Users can filter movies based on:

Search text

Genre

Minimum rating

Year

✔️ Reusable Components

MovieCard

MovieList

FilterBar

SearchBar

✔️ Clean UI & Simple Styling

Built with bootstrap CSS to keep the project easy to understand.

🛠️ Tech Stack

React (useState, useEffect)

JavaScript (ES6+) 

Vite / Create React App (depends on your setup)
